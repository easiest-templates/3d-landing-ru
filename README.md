# NexusFlow — 3D Лендинг

![Preview](https://cdn.poehali.dev/templates/3d-landing-ru/preview.jpg)

Современный лендинг с 3D-элементами и градиентным фоном. Идеален для технологических продуктов и SaaS-решений.

## Quick Start

```bash
npm install
npm run dev
```

## Tech Stack

- React 18 + Vite
- Tailwind CSS
- shadcn/ui
- Framer Motion
